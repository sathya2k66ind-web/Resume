# Resume

A modern, interactive resume project built with React, TypeScript, and Vite.

## Features

- ⚡ Fast development experience with Vite
- 🎨 Styled with Tailwind CSS
- 🎭 Interactive animations with GSAP
- 📦 Three.js integration for 3D graphics
- 📱 Responsive design
- ✨ Beautiful UI components with Lucide React icons

## Tech Stack

- **React** 19.2.3 - UI framework
- **TypeScript** 5.9.3 - Type safety
- **Vite** 7.2.4 - Build tool
- **Tailwind CSS** 4.1.17 - Utility-first CSS
- **GSAP** 3.14.2 - Animation library
- **Three.js** 0.183.2 - 3D graphics
- **Lucide React** 1.7.0 - Icon library

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/sathya2k66ind-web/Resume.git
cd Resume
```

2. Install dependencies
```bash
npm install
```

### Development

Start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:5173`

### Build

Create a production build:
```bash
npm run build
```

### Preview

Preview the production build:
```bash
npm run preview
```

## Configuration

- **Vite Config**: `vite.config.ts` - Build and development configuration
- **TypeScript Config**: `tsconfig.json` - TypeScript compiler options
- **Tailwind CSS**: Configured via `@tailwindcss/vite` plugin

## Features Included

- Path alias support (`@` maps to `src/`)
- Single file build output with `vite-plugin-singlefile`
- Tailwind CSS integration for rapid styling
- Type-safe development with TypeScript
